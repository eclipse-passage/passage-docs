Run Floating License Server
===

Prerequisites
---
 - Java version: 11 (developed and tested on AdoptOpenJDK 11.0.8.10)

Cli Options
---
  -server.port=8090
  [lic residence folder]
  
Cli commands 
---
Lifecycle commands: 
 - start 
 - stop 
 - restart
 
FLS licensing commands: 
 - licstatus 
 - licrequest
